# widgets

Discord widgets tutorial, watch video [here](https://youtu.be/Yq0vDExfVC8) on how to use this repository

# How to use:
1. update the config with correct data
2. run 
```js
node server.s
```
2. make the widget etc (follow the video)
3. open the website `http://localhost:3000`, and paste the generated data fields as json, and your discord id and click submit.

# To note:
- If there is no data fields, put `{"data":{}}` in the json text area.
- You must do the identity part, or else others wont see it.
- Others cant use your own widget.
